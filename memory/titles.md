## 五、标题学习规则

| 触发时机 | 学习内容 | 记录位置 |
|---------|---------|---------|
| 每次生成标题备选 | 全部候选存入风格分区 | AI生成备选库 |
| 用户从备选中选择 | 记录选择偏好，标记★ | 效果档案 + 偏好画像 |
| 用户修改了AI标题 | 分析修改方向 | 效果档案 + 偏好画像 |
| 用户自己写标题 | 学习用户的标题风格 | 偏好画像 |
| 用户反馈效果好/差 | 记录效果数据 | 效果档案 |
| 积累5+条同类型高效 | 提炼高效模式 | 高效标题模式库 |

### 数据积累阈值
- 5篇以上：初步判断用户的标题类型偏好
- 10篇以上：可以提炼2-3个高效标题模式
- 20篇以上：形成完整的用户标题偏好画像
- 50篇以上：标题生成质量应显著优于初始状态
