# 素材库

> 收集高质量的案例、数据、故事、类比等可复用素材。写作前扫描获取灵感，写作中有选择性地嵌入。

## 格式

```
[M-XXX] [素材摘要]
  类型：案例 / 数据 / 故事 / 类比 / 金句观点 / 引述 / 标题技巧
  来源：[文章名/网站/书籍] (日期)
  赛道：[分类]
  标签：[关键词，方便检索]
  质量分：1-5（5最高）
  使用次数：N
  使用场景：[适合什么类型的文章]
```

## 写入规则
- 只收录质量分 ≥ 3 的素材
- 必须标注清晰来源（无来源的不入库）
- 每条素材不超过150字（长的摘取核心部分）
- 引用外部金句时存入本库第三节，不存入 golden-sentences.md

## 使用规则
- 写作前根据当前话题检索匹配素材
- 优先使用质量分高且使用次数少的素材
- 嵌入素材时必须自然融合，不能生硬拼接
- 使用后更新 use_count

## 容量上限：200条
- 超6个月未使用的移到"冷素材区"
- 质量分 < 3 的素材定期清理
