# 观点与立场库

> 记录用户的核心观点、价值判断和立场倾向。这些观点会在后续创作中被自然地融入文章，让每篇文章都带有作者的思想印记。

## 格式

```
- [V-XXX] [观点内容]
  来源：[对话/文章标题] (日期)
  话题：[所属话题分类]
  立场强度：强烈支持 / 支持 / 中立引用 / 谨慎 / 反对 / 强烈反对
  确认状态：已确认 / 待确认（默认）
  最后确认：YYYY-MM-DD（如已确认）
  使用场景：[适合在什么类型的文章中使用]
```

## 立场强度说明

| 强度 | 含义 | 写入条件 |
|------|------|---------|
| 强烈支持 | 用户明确表达强烈认同 | 用户说"我完全同意"/"这是我的核心信念" |
| 支持 | 用户表达认同倾向 | 用户表达赞同但未强调 |
| 中立引用 | 从范文中提取，非用户原创 | 默认值（范文提取时） |
| 谨慎 | 用户有保留地提及 | 用户说"可能吧""不一定" |
| 反对 | 用户明确不认同 | 用户说"我不这么看""这有问题" |
| 强烈反对 | 用户强烈否定 | 用户表达明确反感 |

## 维护规则
- 不确定立场超3个月未确认的 → 移入底部「待确认区」
- 用户在对话中再次提到某观点 → 更新立场强度和确认状态
- 相互矛盾的观点 → 同时保留，标注冲突标记 ⚠️
