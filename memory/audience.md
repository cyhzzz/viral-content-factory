# 读者画像

> 目标读者的特征、偏好和行为习惯。帮助创作时精准把握"写给谁看"。

## 基础画像

```yaml
# 人口统计
age_range: 待了解        # 如：25-40岁
gender_ratio: 待了解    # 如：男60% / 女40%
location: 待了解        # 如：一二线城市为主
occupation: 待了解      # 如：互联网/金融/职场人
income_level: 待了解    # 如：月收入15K+

# 心理特征
core_pain: 待了解       # 最大痛点是什么
core_desire: 待了解     # 最渴望得到什么
fear_factor: 待了解     # 担心/焦虑什么
value_alignment: 待了解 # 认同什么价值观

# 阅读习惯
preferred_length: 待了解    # 喜欢多长的文章
reading_time: 待了解       # 通常什么时候读
reading_device: 待了解     # 手机为主 / 电脑为主
share_behavior: 待了解    # 什么内容会转发
```
