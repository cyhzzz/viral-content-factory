# 金句库

> 收集每次创作中产生的优质金句，按主题分类，供后续文章复用。

## 使用说明
- 每次创作后，将文章中的精彩金句摘录于此
- 按主题分类，标注出处文章和风格标签
- 标记使用次数，避免高频重复（同一条金句不超过3次）
- 超过100条时，淘汰使用效果不佳的

## 金句格式

```
- [G-XXX] 「金句内容」
  出自：《文章标题》(日期)
  风格：犀利 / 温暖 / 幽默 / 哲理 / 朴实 / 激励 / 反讽（可多选，用+连接）
  句式：对比式 / 反转式 / 定义式 / 条件式 / 排比式 / 类比式 / 反问式
  话题：[所属话题]
  已用：N 次
  关联观点：V-XXX（如有）
```

## 风格标签体系

| 标签 | 特征 | 典型句式 |
|------|------|---------|
| 犀利 | 一针见血、直击本质 | "所谓XX，不过是YY的遮羞布" |
| 温暖 | 治愈、共情、柔软有力量 | "没关系，XX本来就是YY的过程" |
| 幽默 | 轻松、自嘲、反差萌 | "XX这件事，说白了就是YY" |
| 哲理 | 深度思考、洞察本质 | "XX的反面不是YY，而是ZZ" |
| 朴实 | 大白话、接地气、真实感 | "说到底，XX就是YY" |
| 激励 | 鼓舞、正能量、行动导向 | "与其XX，不如YY" |
| 反讽 | 反话正说、黑色幽默 | "XX当然是对的，毕竟YY嘛" |

## 创作时金句策略

### 路径一：直接复用旧金句
- 金句库中有高度匹配的金句
- 使用次数 < 3 次（避免审美疲劳）
- 风格与当前文章一致
- 可微调后复用（换主语/调语序），保留核心表达

### 路径二：学习风格生成新金句
- 无匹配现成金句时，从同风格金句中分析句式特征
- 基于3-5条同风格金句提炼模式
- 生成3条候选，选最贴合的1-2条使用
- 新金句自动入库
