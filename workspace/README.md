# workspace/ — 爆款智坊工作空间

> 本目录由 Init Round 1 自动创建，每个子目录用途明确，帮助你有序管理内容创作全流程。

## 目录结构

```
workspace/
├── README.md          # 本文件
├── drafts/            # 草稿区
├── published/        # 已发布区
├── assets/           # 素材资源
├── research/         # 调研笔记
├── templates/        # 自定义模板库
└── archive/          # 历史归档
```

## 各目录用途

### drafts/ — 草稿区

存放正在创作中的文章初稿和修改版本。建议命名格式：`{date}_{title}_draft.md`

### published/ — 已发布区

存放已发布的历史文章。按月份或主题子目录整理，便于回顾和复用。

### assets/ — 素材资源库

```
assets/
├── covers/           # 封面图（3:4比例，建议尺寸 1080×1440）
├── illustrations/    # 内文配图（16:9或1:1，建议尺寸 900×675 或 675×675）
├── charts/           # 数据图表（建议 SVG 或 PNG 透明背景）
└── screenshots/      # 截图素材（手机截图、界面截图等）
```

### research/ — 调研笔记

```
research/
├── topics/           # 选题调研（每次创作的背景调研、竞品分析）
├── competitors/      # 对标账号追踪（标题/结构/风格拆解）
└── hotspots/         # 热点素材（热搜截图、新闻链接、话题背景）
```

### templates/ — 自定义模板库

存放你自己的写作模板。每次创作可选择基础框架或自定义模板。

### archive/ — 历史归档

长期归档：已发布且不再活跃维护的文章、特殊活动素材等。

## 使用建议

1. **草稿命名**：`{日期}_{平台}_{标题}_draft.md`，如 `20260409_wechat_AI投资指南_draft.md`
2. **素材命名**：`{类型}_{日期}_{描述}.{ext}`，如 `cover_20260409_AI主题.jpg`
3. **定期整理**：建议每月末整理一次 drafts/，将完成稿移至 published/，过时素材移至 archive/
