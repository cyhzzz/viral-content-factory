# 作者定位分析框架

> 本文档是风格学习流程 Phase 2 的作者定位识别部分

---

## 作者定位提取

### 身份标签

**识别维度**：

| 标签 | 选项 |
|:---:|:---|
| 职业身份 | 券商投顾 / 基金经理 / 散户 / 自媒体 / 机构 |
| 内容身份 | 专业分析师 / 科普作者 / 实战派 / 经验分享者 |
| 机构背景 | 申万宏源 / 中信证券 / 个人 / 其他机构 |

**提取格式**：
```markdown
## 身份标签
- 职业身份：{选项}
- 内容身份：{选项}
- 机构背景：{选项}
```

---

### 内容定位

**识别维度**：

| 维度 | 选项 |
|:---:|:---|
| 专业程度 | 极专业（需背景知识）/ 中等 / 通俗易懂 |
| 目标受众 | 机构投资者 / 高净值人群 / 大众投资者 / 小白 |
| 内容类型 | 纯分析 / 带操作建议 / 带营销转化 / 纯科普 |

**提取格式**：
```markdown
## 内容定位
- 专业程度：{选项}
- 目标受众：{选项}
- 内容类型：{选项}
```

---

### 读者画像

**识别维度**：

| 维度 | 描述 |
|:---|:---|
| 读者特征 | {年龄/职业/投资经验等} |
| 读者痛点 | {读者关心什么问题} |
| 读者期待 | {读者希望从文章中得到什么} |

**提取格式**：
```markdown
## 读者画像
- 读者特征：{描述}
- 读者痛点：{描述}
- 读者期待：{描述}
```

---

### 平台差异分析（跨平台录入时触发）

**识别维度**：

同一作者在不同平台的内容差异：

| 平台 | 语气正式度 | 段落长度 | 核心差异 |
|:---:|:---:|:---:|:---|
| 微信公众号 | 4/5 | 300字 | {描述} |
| 小红书 | 2/5 | 80字 | {描述} |
| 微博 | 1.5/5 | 140字 | {描述} |

**提取格式**：
```markdown
## 平台差异分析

| 平台 | 语气正式度 | 段落长度 | 核心差异 |
|:---:|:---:|:---:|:---|
| {平台} | {N}/5 | {字数} | {描述} |
| ... | ... | ... | ... |

**差异总结**：
- {总结}
```

```markdown

## 身份标签
- 职业身份：{描述}
- 内容身份：{描述}
- 机构背景：{描述}

## 内容定位
- 专业程度：{描述}
- 目标受众：{描述}
- 内容类型：{描述}

## 读者画像
- 读者特征：{描述}
- 读者痛点：{描述}
- 读者期待：{描述}

## 平台差异（仅跨平台录入时）
{平台差异表格}

## 写作目标
- **作者希望读者感受到**：{描述}
- **作者希望读者得到**：{描述}
- **作者的独特价值主张**：{描述}
```
