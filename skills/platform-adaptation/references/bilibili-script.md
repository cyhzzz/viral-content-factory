# B站中视频脚本规范

> 本文档是 B站（哔哩哔哩）中视频脚本的完整指南，与抖音/快手短视频脚本有本质区别。

---

## B站生态解读

### 平台定位

B站是以 **Z世代（18-25岁）为核心用户** 的中视频社区。与抖音"刷完即走"不同，B站用户把视频当"内容"消费——他们会收藏、投币、反复观看、到评论区深度讨论。

| 维度 | 抖音/快手 | B站 |
|------|----------|-----|
| 用户心态 | 杀时间 | 求知/娱乐/社交 |
| 内容期待 | 刺激爽点 | 信息增量+深度 |
| 完播率基准 | 30-50% | 40-60%（越高越好） |
| 互动深度 | 点赞为主 | 弹幕+评论+投币 |
| 推荐逻辑 | 爆款优先 | 标签匹配+社区信任 |

### 核心指标优先级

1. **完播率**（最重要）：反映内容质量，决定是否推更大流量池
2. **弹幕密度**：引发讨论=算法加分=二次传播
3. **投币收藏**：高互动=账号权重提升
4. **评论质量**：有深度的评论区可吸引新用户驻足

## B站算法机制

### 推荐算法三阶段

```
第一阶段：冷启动（500-1000曝光）
  → 考核：点击率 + 完播率
  → 达标→进入第二阶段

第二阶段：内容标签匹配（1万-10万曝光）
  → 考核：完播率 + 弹幕/评论密度
  → 达标→进入第三阶段

第三阶段：爆款推送（10万+曝光）
  → 考核：互动率（弹幕/评论/投币/收藏）
  → 持续达标→长期推荐
```

### 标签机制

B站推荐依赖 **内容标签 + 用户兴趣标签** 双重匹配。UP主需要在标题、简介、标签中埋入精准关键词：

- **标题**：包含核心关键词（1-2个）
- **简介**：补充相关标签词（3-5个）
- **标签**：选择1个主标签 + 2-3个副标签
- **封面文字**：与标题形成关键词互补

### 分区策略

B站有20+内容分区，常见财经分区：
- **理财/投资**（高变现价值）
- **财经杂谈**
- **商业**
- **科技**

选择分区直接影响初始流量池大小。

## 创作激励计划

B站创作激励按 **播放量 × 互动系数** 计算收益：

| 指标 | 权重 | 说明 |
|------|------|------|
| 播放量 | 1.0x | 基础曝光 |
| 弹幕数 | 5.0x | 互动价值最高 |
| 评论数 | 3.0x | 质量>数量 |
| 点赞数 | 1.5x |  |
| 投币数 | 8.0x | 最高权重 |
| 收藏数 | 4.0x | 用户主动收藏=认可 |

**优化方向**：一条爆款视频（50万播放）配合高弹幕密度，可获得数千元激励。

## 中视频脚本结构模板

B站中视频（1-30分钟）与抖音短视频脚本结构完全不同。抖音靠"3秒钩子"留人，B站靠"信息密度 + 节奏设计"让人看完。

### 完整脚本模板

```markdown
# {视频标题}

## 基本信息

- **时长**：约 {N} 分钟
- **类型**：{口播/动画/混剪/真人出镜+素材}
- **分区**：{理财/财经杂谈/商业/科技}
- **标签**：{核心标签词1} {标签词2} {标签词3}

## 简介（Description）

{50-100字的视频简介，包含：
- 核心看点
- 目标人群
- 关键词埋入
- 引导评论的问题
}

## 分P结构（如适用）

若视频超过8分钟，建议分P：
- P1：{阶段1主题}（约{N}分钟）
- P2：{阶段2主题}（约{N}分钟）
- P3：{阶段3主题}（约{N}分钟）

## 脚本正文

### 开场（0:00-0:30）

**画面**：「{描述}」
**配音**：「{开场白，抛出核心问题或悬念}」
**弹幕引导**：「期待的问题可在评论区打出"1"」

### 第一部分：背景/问题（0:30-3:00）

**画面**：「{描述}」
**配音**：「{交代背景，引出核心问题}」
**信息密度提示**：「每30秒至少1个信息点/数据/观点」

### 第二部分：核心分析（3:00-8:00）

**画面**：「{描述}」
**配音**：「{深度分析，2-3个核心论点}」
**弹幕引导**：「这里可以设置讨论点」

### 第三部分：案例/数据（8:00-12:00）

**画面**：「{描述}」
**配音**：「{具体案例或数据支撑}」
**互动设计**：「你觉得这个案例说明了什么？弹幕聊聊」

### 第四部分：结论+建议（12:00-15:00）

**画面**：「{描述}」
**配音**：「{总结核心观点，给出可操作建议}」
**收藏引导**：「收藏起来，下次用得上」

### 结尾引导（最后30秒）

**画面**：「{描述}」
**配音**：「{引导关注+互动问题+风险提示}」
**投币引导**：「如果对你有帮助，一键三连是对我最大的支持」
```

## 分P策略

### 何时分P

| 视频时长 | 分P建议 | 理由 |
|---------|--------|------|
| <8分钟 | 不分P | 内容体量不需要分割 |
| 8-15分钟 | 可分2P | 节奏段落自然分割 |
| 15-30分钟 | 建议分2-3P | 信息量大，分P降低跳出率 |
| >30分钟 | 必须分P | 超长视频完播率会急剧下降 |

### 分P节奏设计

```
P1：铺垫+问题（总时长的20%）
  → 建立期待，让用户知道会得到什么

P2：核心分析（总时长的50%）
  → 信息密度最高的部分

P3：结论+延伸（总时长的30%）
  → 收尾+可收藏的价值点
```

### 分P标题公式

```
P1：{悬念型标题}（例："为什么现在布局XXX？"）
P2：{解决方案型}（例："3步教你XXXX"）
P3：{总结型}（例："一文讲透XXXX"）
```

### 分P注意事项

- 每P时长尽量均匀（避免某P过短或过长）
- P1结尾留钩子，引导用户点P2
- 多P视频在算法上有轻微劣势（完播率计算复杂），但总时长可更长

## 弹幕互动设计

弹幕是B站的核心互动资产，是用户"参与创作"的方式。好的弹幕设计可以引发连锁讨论。

### 弹幕触发类型

| 类型 | 触发方式 | 示例 |
|------|---------|------|
| 认同型 | 说出用户心声 | "说到心坎里了" |
| 疑问型 | 留出思考空间 | "这里为什么要XXX？" |
| 补充型 | 提供额外信息 | "补充：其实还有XXX" |
| 玩梗型 | 设置可接的梗 | "来了来了" |
| 情绪型 | 触发强烈情绪 | "太真实了/泪目" |

### 弹幕设计技巧

#### 1. 设问式弹幕点

在关键论点后抛出问题，引导用户在弹幕或评论区回答：

```
配音：「所以现在的关键是——我们要判断XXX还是YYY？」
弹幕引导：「判断错的在弹幕打1，对的打2」
```

#### 2. 数据冲击式

用具体数字触发"震惊"弹幕：

```
配音：「2024年Q1，创业板成交额突破万亿——这是三年来首次」
弹幕引导：「见证历史的弹幕扣1」
```

#### 3. 观点争议式

设置可以有不同观点的内容点：

```
配音：「有人说是机会，有人说是陷阱，你怎么看？」
弹幕引导：「机会扣1，陷阱扣2，评论区说说你的判断」
```

#### 4. 情感共鸣式

触及目标群体的共同经历或情感：

```
配音：「相信很多人跟我一样，刚入市的时候也是一脸懵」
弹幕引导：「初代韭菜报到」
```

### 弹幕密度控制

- 理想密度：平均每分钟5-15条弹幕
- 弹幕太少：显得内容冷清
- 弹幕太多：遮挡关键信息
- 在视觉重点区域可关闭弹幕（如数据图表区域）

## 结尾引导与社区运营

### 结尾引导公式

B站结尾不宜太短，需要完成三个任务：

```
★ 核心结论回顾（15秒）
  → 帮助用户梳理今天学到了什么

★ 互动引导（10秒）
  → 提出讨论问题，引导评论区

★ 关注+投币引导（10秒）
  → "一键三连是对我最大的支持"
```

### 评论区运营

发布后30分钟内回复第一批评论，可提升算法推荐权重。

评论引导技巧：
- 抛出一个延伸问题
- 对热门评论进行"楼中楼"回复
- 置顶一条高质量评论

## 涨粉策略

### B站涨粉特点

B站粉丝粘性极高（对比其他平台），但涨粉速度慢于抖音。核心策略是 **"系列化内容 + 深度互动"**。

### 涨粉关键动作

| 动作 | 效果 | 执行方式 |
|------|------|---------|
| 系列视频 | 最高效的涨粉路径 | 做成"第X期"系列，形成追更 |
| 垂直深耕 | 建立账号标签 | 持续在同一分区发布 |
| 弹幕互动 | 增强用户归属感 | 读弹幕/回应弹幕 |
| 动态互动 | 日常维护粉丝 | 发布日常/预告/投票 |
| 合作互推 | 快速破圈 | 与同级别UP主互相推荐 |

### 系列化模板

```
{固定前缀}系列命名
例：「投资必知系列」「A股行业研究系列」「理财避坑系列」

每期结构固定：
P1：开头固定语（建立识别度）
P2：独立主题（每期讲一个具体内容）
P3：结尾固定引导（关联下期预告）
```

## 变现路径

B站提供5种主要变现方式，按难度排序：

### 1. 创作激励（门槛最低）

- 条件：正式会员 + 开通创作激励
- 收益：播放量×互动系数
- 特点：稳定但单价低，适合高频更新

### 2. 充电（粉丝经济）

- 条件：1000+粉丝可申请
- 方式：粉丝月度充电（6元/月起）
- 特点：铁粉付费意愿强，但规模有限

### 3. 直播（实时互动）

- 条件：完成实名认证
- 方式：打赏 + 直播带货 + 直播课程
- 特点：需要稳定粉丝基础，互动性强

### 4. 商业合作/赞助（主流变现）

- 条件：1万+粉丝开始有品牌主动联系
- 方式：定制内容 + 植入 + 联合活动
- 特点：单价高，但需注意内容把控

### 5. 课程/付费内容（高价值变现）

- 条件：需要专业资质或认证
- 方式：B站课堂 + 付费专栏
- 特点：单价最高，但制作成本也最高

### 变现优先级建议

```
粉丝<1万：专注创作激励 + 积累内容
粉丝1-10万：开始接受商业合作
粉丝10万+：可考虑课程/付费内容
```

## B站 vs 抖音核心差异

| 维度 | B站中视频 | 抖音/快手短视频 |
|------|----------|----------------|
| 时长 | 1-30分钟 | 15秒-3分钟 |
| 核心指标 | 完播率+弹幕密度 | 点赞率+完播率 |
| 内容深度 | 需要信息增量 | 情绪刺激即可 |
| 互动形式 | 弹幕+评论+投币 | 点赞+评论 |
| 用户期待 | 学到东西/深度娱乐 | 杀时间/爽一下 |
| 粉丝价值 | 高（粘性强） | 低（易流失） |
| 变现速度 | 慢（需积累） | 快（爆款即变现） |
| 算法逻辑 | 标签匹配+社区信任 | 爆款优先 |
| 标题风格 | 信息量大+悬念 | 情绪化+冲击力 |
| 封面风格 | 信息清晰+标题辅助 | 冲击感+人物特写 |

### 内容策略差异

```
抖音：爆款思维 → 追求单条视频极致传播
B站：账号思维 → 追求粉丝积累和长期价值

抖音：3秒定生死
B站：3分钟才开始（完播率需要慢慢建立）
```

## 质量检查清单

### 脚本层面

- [ ] 时长与内容体量匹配（1分钟约180字，5分钟约900字）
- [ ] 每分钟至少1个信息点/数据/观点
- [ ] 开场30秒有明确的价值承诺（告诉用户会得到什么）
- [ ] 有弹幕互动设计（至少2处）
- [ ] 结尾有核心结论回顾
- [ ] 结尾有互动引导问题
- [ ] 配音口语化，无书面语

### 标签与SEO

- [ ] 标题含核心关键词（1-2个）
- [ ] 简介含补充关键词（3-5个）
- [ ] 选择正确的分区
- [ ] 标签设置合理（1主+2副）

### 合规要求

- [ ] 无具体个股推荐
- [ ] 无承诺收益表达
- [ ] 有风险提示（建议放在结尾）
- [ ] 数据有来源标注（如"据Wind数据"）
- [ ] 无极端词汇（"一定""保证""稳赚"等）

## 反模式与避坑

### ❌ 常见错误

| 错误 | 问题 | 正确做法 |
|------|------|---------|
| 直接搬运抖音脚本 | B站用户厌恶低信息密度内容 | 针对B站重写，增加信息增量 |
| 标题党过度 | B站用户对骗点击容忍度低 | 标题承诺与内容匹配 |
| 忽略弹幕设计 | 损失算法加权 | 主动设计弹幕触发点 |
| 单期内容孤立 | 涨粉效率低 | 做系列化内容 |
| 结尾太短 | 浪费变现机会 | 结尾至少30秒完整引导 |
| 不回复评论 | 损失社区信任 | 发布后30分钟内回复 |

### ⚠️ 特殊注意

- B站用户对"广告感"非常敏感，商业合作需要软植入
- 财经内容需格外注意合规，避免暗示性投资建议
- 数据引用需标注来源，避免被质疑
- 封面不要过度夸张，B站用户审美偏好真实感

## B站脚本示例（完整结构）

```markdown
# 2024年，还能靠定投赚钱吗？数据说话

## 基本信息

- **时长**：约 12 分钟
- **类型**：口播 + 数据图表
- **分区**：理财/投资
- **标签**：定投 基金 投资策略 数据分析

## 简介（Description）

很多人在问，现在还适合定投吗？本期用数据说话——
分析了近10年股债数据，告诉你什么情况下定投有效，
什么情况下可能是陷阱。结尾有实操建议，收藏起来。

## 分P结构

- P1：定投还能不能做？（约3分钟）
- P2：10年数据告诉你真相（约5分钟）
- P3：什么策略才是对的（约4分钟）

## 脚本正文

### P1：开场

**画面**：「真人出镜，背景为书房」
**配音**：「你是不是也在纠结，现在还要不要继续定投？」
**弹幕引导**：「正在定投的扣1，准备开始的扣2」

### P1：问题铺垫

**画面**：「数据图表：近5年定投收益」
**配音**：「2021年高点入场的朋友，现在大概率还在亏。」
「但这不是定投的错——是方法的问题。」

### P2：核心数据

**画面**：「逐帧展示数据图表」
**配音**：「我们回测了2014-2024年的数据——」
「沪深300指数，每月定投1000元。」
「结果出乎意料……」
**弹幕引导**：「觉得定投有效的弹幕扣1，觉得没用的扣2」

### P3：结论

**画面**：「真人出镜，指向提纲」
**配音**：「结论就三句话——」
「第一，定投有效，但需要时间。」
「第二，择时入场很重要。」
「第三，股债配比是核心。」
「具体的配置方案，我已经整理好了——」
「评论区留言'配置'，我把模板发给你。」
**收藏引导**：「觉得有用的收藏起来，下次选基的时候用得上」

### 结尾引导

**画面**：「三连按钮特写」
**配音**：「本期数据都放在简介里了，可以直接去看。」
「如果你也有定投的问题，评论区聊聊。」
「一键三连是对我最大的支持，我是XX，下期见。」
「市场有风险，投资需谨慎。」

## 配乐建议

风格：轻音乐，节奏舒缓，不抢人声
节奏：每分钟80-90拍，不适合动感BGM
```
