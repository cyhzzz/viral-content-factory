# 今日头条规范

> 本文档是多平台改写流程 Step 6.5 的详细操作手册

---

## 今日头条输出要求

### 格式要求

| 要求 | 说明 |
|-----|------|
| 文件格式 | Markdown（.md） |
| 段落 | 短段落，每段 ≤ 3句 |
| 关键词 | 前3段必须出现 |
| 标题 | 与原标题不同（情绪化） |
| 算法友好 | 信息密度高 |

---

## 今日头条特点

- **算法推荐**：头条靠算法分发，关键词密度影响推荐量
- **短阅读**：用户多为碎片化阅读，段落要短
- **情绪化**：标题和开头要有情绪钩子
- **信息量**：单位字数信息量要高

---

## 标题规范

### 头条标题特点

| 特点 | 说明 | 示例 |
|-----|------|------|
| 情绪化 | 有情绪词 | A股暴涨！这件事太重要了 |
| 悬念 | 引发好奇 | 2024年，股市最大的机会在这里？ |
| 数字 | 具体可感 | 3个信号告诉你，该关注这个板块了 |
| 反常识 | 与认知相悖 | 为什么老手都在减仓，新手却在抄底？ |

### 标题规则

- 字数：15-30字
- 必须含情绪词或悬念元素
- 必须含核心关键词（利于算法识别）
- 避免过度标题党

---

## 正文规范

### 段落规则

- 每段 ≤ 3句
- 关键信息放段首
- 每段首句必须是完整句（算法抓取）

### 关键词密度

- 前3段必须出现核心关键词
- 全文关键词出现 ≥ 3次
- 关键词自然分布，不堆砌

### 头条句式

```
开头（必须钩子）：
- 刚刚，传来重磅消息...
- A股今天这个信号，必须重视...
- 2024年，这个板块可能要爆发了...

信息密度：
- 每句话必须有信息增量
- 避免重复表达
- 少用过渡句
```

---

## 今日头条结构

```markdown
# {情绪化标题}

{第1段：钩子+核心信息，关键词出现}
（空行）
{第2段：核心观点，关键词出现}
（空行）
{第3段：支撑论据，关键词出现}
（空行）
{第4段：数据/案例}
（空行）
{第5段：分析解读}
（空行）
{第6段：结论+风险提示}
```

## 今日头条示例

```markdown
# A股暴涨！这个信号必须重视

刚刚，A股三大指数集体大涨，创业板指涨超3%。成交额突破1.5万亿，市场情绪明显回暖。

这个消息太重要了。

今天的大涨，不是偶然。成交量的放大，说明有增量资金进场。而增量资金的方向，往往决定了接下来行情的走向。

从板块来看，新能源、科技涨幅居前。这两个方向，也是我们之前反复提示关注的。

怎么看后市的持续性？关键看成交量能否维持在1.5万亿以上。如果能，大概率还有机会；如果不能，需要谨慎。

操作上，可以关注有业绩支撑的科技龙头。但要注意，市场有风险，投资需谨慎。
```

---

## 头条合规要求

### 必须包含

- 风险提示
- 数据来源标注

### 禁止表达

- ❌ "一定涨"、"保证赚钱"
- ❌ 具体个股推荐
- ❌ 极端词汇

---

## 质量检查清单

- [ ] 标题情绪化，与原标题不同
- [ ] 前3段有关键词
- [ ] 每段 ≤ 3句
- [ ] 信息密度高
- [ ] 风险提示完整
- [ ] 无个股推荐
- [ ] 无极端词汇
