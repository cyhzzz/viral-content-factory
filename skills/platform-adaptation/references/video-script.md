# 短视频脚本规范

> 本文档是多平台改写流程 Step 6.6 的详细操作手册

---

## 短视频脚本输出要求

### 格式要求

| 要求 | 说明 |
|-----|------|
| 文件格式 | Markdown（.md） |
| 时长 | 1分钟脚本 ≈ 180字 |
| 结构 | 开场钩子 + 正文 + 结尾引导 |
| 分镜 | 镜头编号 + 画面描述 + 配音稿 |

---

## 短视频脚本结构

```markdown
# {视频标题}

## 基本信息

- **时长**：约 {N} 秒
- **类型**：{口播/动画/真人出镜/混剪}
- **平台**：抖音/快手/视频号

## 分镜脚本

### 镜头1：开场钩子

**画面**：{描述}
**配音**：「{3秒内说完的核心钩子}」

### 镜头2：引入

**画面**：「{描述}」
**配音**：「{引出话题，10秒}」

### 镜头3：核心内容1

**画面**：「{描述}」
**配音**：「{关键信息，15秒}」

### 镜头4：核心内容2

**画面**：「{描述}」
**配音**：「{关键信息，15秒}」

### 镜头5：总结

**画面**：「{描述}」
**配音**：「{总结+行动引导，10秒}」

## 字幕参考

{关键字幕内容}

## 配乐建议

{风格建议}
```

---

## 开场钩子规范

### 3秒钩子类型

| 类型 | 示例 |
|-----|------|
| 疑问句 | 什么是ETF？为什么新手都在买？ |
| 数字 | 3个信号，告诉你该不该买基金 |
| 反常识 | 老手都在减仓，新手却在抄底？ |
| 感叹句 | A股今天这个信号，太重要了！ |
| 场景句 | 你是不是也在纠结，现在该不该买基金？ |

### 钩子规则

- 3秒内必须说完
- 必须含核心关键词
- 必须引发好奇或共鸣

## 分镜规范

### 分镜格式

```markdown
### 镜头{N}：{镜头名称}

**画面**：「{具体画面描述}」
**配音**：「{配音稿}」
**时长**：「{秒数}」
```

### 画面描述要求

每个画面必须包含：
1. **场景**：室内/室外/具体地点
2. **人物**：（如适用）描述人物动作/表情
3. **道具**：白板/PPT/产品/图表等
4. **文字**：是否叠加字幕

### 配音稿要求

- 口语化
- 短句为主
- 每句不超过15字
- 关键数据/结论加重音标注（★）

---

## 配音稿口语化规则

| 书面语 | 口语化 |
|-------|-------|
| 因此 | 所以、那也就是说 |
| 然而 | 但是、不过 |
| 根据数据显示 | 数据告诉我们、你看 |
| 综上所述 | 总的来说、最后 |
| 投资需谨慎 | 大家要注意、市场有风险 |

---

## 结尾引导规范

### 结尾引导类型

| 类型 | 示例 |
|-----|------|
| 关注引导 | 关注我，每天分享投资干货 |
| 评论引导 | 觉得有用的话，评论区聊聊 |
| 收藏引导 | 收藏起来，慢慢看 |
| 行动引导 | 想了解更多，首页有详细教程 |

### 结尾公式

```
★ 总结核心观点（5秒）
+ 行动引导（5秒）
= 10秒结尾
```

```markdown
[0:55-1:00] 关注我，下期讲讲怎么用ETF做资产配置
```

---

## 短视频合规要求

### 必须包含

- 风险提示（配音或字幕）
- 无具体个股

### 禁止表达

- ❌ "跟着买一定赚"
- ❌ 具体个股
- ❌ 极端词汇
- ❌ 承诺收益

### 风险提示位置

风险提示可在：
- 结尾字幕：市场有风险，投资需谨慎
- 全程口播提醒

## 短视频脚本示例

```markdown
# 一分钟搞懂ETF

## 基本信息

- **时长**：约 60 秒
- **类型**：口播 + 字幕动画
- **平台**：抖音/视频号

## 分镜脚本

### 镜头1：开场钩子

**画面**：「特写：手指点向镜头」+
**配音**：「什么是ETF？为什么新手都在买？」
**时长**：「3秒」

### 镜头2：问题引入

**画面**：「真人出镜，背景为办公桌」+
**配音**：「你是不是也在纠结，现在该不该投资基金？」
**时长**：「5秒」

### 镜头3：ETF是什么

**画面**：「动画：一篮子水果 = ETF概念」+
**配音**：「ETF就是让普通人，一键配置一篮子股票。简单说，就像买了个套餐，不用自己一样样挑。」
**时长**：「15秒」

### 镜头4：ETF优势

**画面**：「动画：三个优势图标」+
**配音**：「★第一，省心。不用研究个股。」+
「★第二，省钱。费率比主动基金低。」+
「★第三，★能避免追涨杀跌。因为你是一次性买入，不会择时。」
**时长**：「20秒」

### 镜头5：适合人群

**画面**：「真人出镜」+
**配音**：「如果你没时间研究市场，又想分散风险，ETF是个不错的选择。」
**时长**：「10秒」

### 镜头6：结尾引导

**画面**：「手指向右侧 + 关注按钮」+
**配音**：「关注我，下期教你用ETF做资产配置。市场有风险，投资需谨慎。」
**时长**：「7秒」

## 字幕参考

```
[0:00-0:03] 什么是ETF？为什么新手都在买？
[0:05-0:10] 你是不是也在纠结，现在该不该投资基金？
[0:10-0:25] ETF就是让普通人，一键配置一篮子股票
[0:25-0:45] ★省心 ★省钱 ★避免追涨杀跌
[0:45-0:55] 关注我，下期教你用ETF做资产配置
[0:55-0:60] 市场有风险，投资需谨慎
```

## 配乐建议

风格：轻快金融风，节奏感强，不干扰人声
```
