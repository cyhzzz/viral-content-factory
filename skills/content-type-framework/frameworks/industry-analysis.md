# 行业分析框架

**适用场景**：单一行业/板块/概念/产业链的深度分析，投资逻辑拆解，ETF基金推荐逻辑。
**核心特征**：产业逻辑、竞争格局、产业链结构、长期趋势、估值判断
**篇幅建议**：2000-3000 字（单行业深度）/ 3000-4000 字（产业链/跨行业对比）
**内容类型路由权重**：Primary

---

## 段落结构

| 段落 | 功能 | 固定开头语 | 字数范围 |
|:---:|:---|:---|:---:|
| 第1段 | 行业全景 | {行业名}是... | 150-200字 |
| 第2段 | 产业链解构 | 从产业链视角... | 200-300字 |
| 第3段 | 竞争格局 | 竞争格局上... | 200-300字 |
| 第4段 | 驱动逻辑 | 核心驱动... | 200-300字 |
| 第5段 | 估值与风险 | 估值层面... | 150-200字 |
| 第6段 | 投资策略 | 操作上... | 150-200字 |

> 精简版（2000字内）可合并为5段：全景→产业链→竞争→驱动→策略

---

## 各段落写作规范

### 第1段：行业全景

**本段核心任务**：用2-3个关键数字+一句话定义，让读者快速建立对行业的认知框架。

**必备要素**：
- 行业一句话定义（是做什么的，服务什么需求）
- 行业总产值/市场规模（最新数据）
- 行业所处周期位置（导入期/成长期/成熟期/衰退期）
- 当前行业最核心的矛盾

**可选要素**：
- 行业增速（ CAGR ）
- 政策定位（是否为国家战略/鼓励/限制）
- 主要参与者数量级

**金句方向**：用一句话点明行业本质，如"白酒行业的本质不是酒，是社交货币"

**数据要求**：
```
必须：行业规模（亿元/万亿）、行业增速（CAGR）
可选：企业数量、市场集中度（CR3/CR5）
```

---

### 第2段：产业链解构

**本段核心任务**：把行业的价值链条拆解清楚，让读者理解钱在哪里、话语权在哪里。

**必备要素**：
- 产业链上中下游划分（每个环节的核心玩家类型）
- 各环节的价值分布（哪个环节最赚钱，占比多少）
- 关键环节的定价权归属（谁说了算）
- 上下游的博弈关系

**分析框架**：
```
上游（原材料/零部件）
    ↓
中游（制造/加工/服务）
    ↓
下游（渠道/品牌/终端用户）
```

**可选要素**：
- 产业链的价值流动方向
- 关键卡脖子环节
- 产业链迁移趋势（是否向海外转移）

**金句方向**：用一句话点明产业链的核心特征，如"这个行业最值钱的部分不在制造，而在渠道"

**数据要求**：
```
可选：各环节毛利率对比、龙头公司市值/营收
```

---

### 第3段：竞争格局

**本段核心任务**：拆解行业竞争结构，让读者理解为什么是这个格局，谁在赢。

**必备要素**：
- 市场集中度（CR3/CR5/CR10）
- 竞争格局类型（完全竞争/寡头/垄断/双寡头）
- 龙头公司的竞争优势（规模/品牌/技术/渠道/成本）
- 新进入者的威胁程度

**分析框架**：
```
波特五力简化版：
- 现有竞争者：CR3/5、市场份额分布
- 新进入者：壁垒高低、进入成本
- 替代品：替代威胁程度
- 上游议价：原材料价格波动影响
- 下游议价：客户黏性、转换成本
```

**可选要素**：
- 近年竞争格局变化（谁在崛起，谁在衰落）
- 国际化竞争（国内龙头 vs 海外巨头）

**金句方向**：用一句话描述竞争格局的本质，如"这个行业本质是'赢家通吃'，第二名活得都很艰难"

**数据要求**：
```
必须：市场集中度数据（CR3/5）
可选：龙头公司市场份额历年变化
```

---

### 第4段：核心驱动逻辑

**本段核心任务**：揭示驱动行业未来增长/衰退的根本力量，这是全文最重要的一段的。

**必备要素**：
- 2-3个核心驱动力（政策/技术/需求/供给/人口/汇率等）
- 每个驱动力的逻辑链条（为什么会持续）
- 驱动力的时间窗口（3年/5年/10年维度）
- 当前最关键的边际变化（最近什么在变）

**可选要素**：
- 行业周期性特征（和经济周期的关系）
- 季节性特征（如白酒的节假日效应）
- 政策周期的影响

**驱动力类型参考**：
```
需求端：人口结构变化、消费升级/降级、技术替代新需求
供给端：产能扩张/收缩、技术革新、成本变化
政策端：监管松紧、政策补贴/限制
资金端：外资流入/流出、机构持仓变化
```

**金句方向**：用一句话概括核心驱动，如"这个行业真正的大周期，是人口老龄化带来的二十年黄金期"

**数据要求**：
```
必须：历史增速数据、政策力度数据
可选：预测模型数据（如有权威机构预测）
```

---

### 第5段：估值与风险

**本段核心任务**：给读者一个客观的估值参照系，同时坦诚风险。

**必备要素**：
- 当前估值水平（PE/PB/PS等，与历史中枢比）
- 估值贵还是便宜的理由（3个支撑/压制因素）
- 主要风险（2-3个，按重要性排序）
- 风险的触发条件（什么时候风险会兑现）

**估值参照系**：
```
历史分位：当前估值处于历史X%分位
行业横向对比：行业平均PE vs 沪深300 PE
国际对比：A股 vs 海外龙头估值（如有）
```

**金句方向**：用一句话表达对风险的理解，如"最大的风险不是下跌，而是你买入的逻辑本身就不存在了"

**数据要求**：
```
必须：当前PE/PB、估值分位数
可选：龙头公司一致预期EPS增速
```

---

### 第6段：投资策略

**本段核心任务**：给出可操作的策略建议，包括标的选择、买入方式、时间窗口。

**必备要素**：
- 投资方向建议（个股/ETF/行业基金）
- 买入方式（一次性/定投/分批）
- 适合的人群（风险偏好/资金性质/持有期限）
- 持有期限建议（短期/中期/长期）
- 止损条件（在什么情况下应该退出）

**禁止**：
- 不能说"推荐买入XX股票"（可以说"关注XXETF"）
- 不能说"XX股票的目标价是XX元"

**可选要素**：
- 相关ETF代码（如有）
- 行业主动管理基金推荐逻辑
- 当前是否是好的买入时机

**金句方向**：用一句话给出一个可验证的标准，如"当XX指标超过X时，意味着这波行情可能还没走完"

**数据要求**：
```
可选：相关ETF代码、费率对比
```

## 标题规范

**结构**：
```
【行业深度】{行业名}深度报告：{核心观点标题}
【板块解读】{行业名}：{一个独特的视角}
【ETF研究】{行业名}ETF：{投资逻辑一句话}
```

**字数**：主标题 15-30 字

**情绪**：冷静客观，以数据和研究说话，不煽情

**标题类型选择**：

| 类型 | 示例 |
|-----|------|
| 格局型 | 【行业深度】从产业链逻辑，看光伏行业的投资价值 |
| 风险型 | 【行业深度】CXO赛道泡沫出清了吗？ |
| 对比型 | 【行业深度】白酒vs酱油：谁才是A股最好的生意？ |
| 预测型 | 【行业深度】消费电子的下一个五年，机会在哪里？ |

## 数据使用规范

### 必需数据

| 数据类型 | 精度要求 | 表达方式 |
|---------|---------|---------|
| 行业规模 | 精确到亿元 | "2.8万亿元" |
| 行业增速 | 精确到0.1% | "年复合增速12.5%" |
| 市场份额 | 精确到0.1% | "CR3超过65%" |
| 估值数据 | 精确数值 | "PE-TTM为28.6倍" |

### 数据来源标注

| 数据类型 | 标注格式 |
|---------|---------|
| 行业协会数据 | "根据中国XX行业协会数据" |
| 上市公司财报 | "根据公司财报披露" |
| 第三方研究 | "根据XX证券研究报告" |
| 国家统计局 | "根据国家统计局数据" |

## 句式规范

### 推荐使用的句式

**产业链描述句**：
```
XX处于产业链的核心位置，掌握着YY的定价权
从XX到YY，价值链条的分配比例约为A:B:C
YY是整个产业链的"咽喉"，因为ZZ
```

**竞争格局描述句**：
```
行业呈现"强者恒强"的格局，CR3市场份额过去5年提升了X个百分点
第二梯队与龙头之间的差距在扩大，而不是缩小
新进入者面临的壁垒主要是ZZ，年资本开支门槛约X亿元
```

**投资逻辑描述句**：
```
核心逻辑是XX的YY趋势确定性高，持续时间预计Z年
边际变化在于XX，这使得原本YY的逻辑变成了ZZ
当前的主要矛盾是XX，但随着YY变化，主要矛盾会切换到ZZ
```

### 禁止使用的句式

```
❌ XX行业必将迎来大爆发，现在买入就是捡钱
❌ XX公司是绝对龙头，闭眼买入就能赚钱
❌ XX行业的增长是没有上限的
❌ XX股票看到XX元的目标价
```

## 词汇规范

### 行业分析高频词

| 术语 | 含义/用法 |
|-----|---------|
| 渗透率 | 新产品/服务在目标市场的占比，衡量成长阶段 |
| CAGR | 年复合增长率 |
| 集中度(CR3/5/10) | 行业前3/5/10名企业的市场份额之和 |
| 护城河 | 阻挡竞争对手的优势，持续性如何 |
| 供给侧改革 | 淘汰落后产能，提高行业集中度 |
| 进口替代 | 国内产品替代进口产品的过程 |
| 产能周期 | 扩产→过剩→出清的周期性规律 |
| 估值中枢 | 历史平均估值水平 |
| 戴维斯双击 | 业绩和估值同时提升，股价大涨 |

### 谨慎词

```
✅ 推荐：大概率、或、可能、值得关注、仍需观察
❌ 禁用：必将、确定、一定、翻倍、茅台（个股）
```

## 风险提示规范

**位置**：文章最后一段结尾，或单独成段

**格式**：
```
本内容仅供参考，不构成投资建议。投资者据此操作，风险自担。
市场有风险，投资需谨慎。
```

**合规检查项**：
- [ ] 无个股直接推荐（可提ETF/行业基金）
- [ ] 无目标价/估值预测（可以说"估值处于历史低位"）
- [ ] 风险提示完整
- [ ] 投资建议有持有期限和适用人群

---

## 质量检查清单

**结构层面**：
- [ ] 第1段包含行业定义+规模+所处周期
- [ ] 第2段包含产业链三环节+价值分布
- [ ] 第3段包含竞争格局+集中度数据
- [ ] 第4段包含2-3个核心驱动力+时间窗口
- [ ] 第5段包含估值水平+主要风险
- [ ] 第6段包含投资方向+止损条件

**逻辑层面**：
- [ ] 产业链逻辑清晰，读者能理解钱在哪里
- [ ] 竞争格局分析有数据支撑
- [ ] 驱动逻辑的因果链条完整
- [ ] 风险识别客观，不回避不利因素

**合规层面**：
- [ ] 无个股直接推荐
- [ ] 无目标价预测
- [ ] 有止损条件
- [ ] 风险提示格式正确

**传播层面**：
- [ ] 有至少1句可独立传播的金句
- [ ] 产业链/竞争格局有具体数字
- [ ] 投资逻辑有独特视角，不是泛泛而谈
