# 投教营销框架

**适用场景**：投资基础知识普及（开户/基金/理财/指标讲解）、工具使用介绍、券商产品/服务推广、新客权益介绍。
**核心特征**：通俗易懂、场景化、行动引导、转化导向、信任建立
**篇幅建议**：800-1200 字（单知识点）/ 1500-2000 字（系统性教程）
**内容类型路由权重**：Primary

---

## 段落结构

| 段落 | 功能 | 固定开头语 | 字数范围 |
|:---:|:---|:---|:---:|
| 第1段 | 痛点/场景切入 | 你是不是也... | 100-150字 |
| 第2段 | 问题本质 | 其实，XX是这样的... | 150-200字 |
| 第3段 | 核心知识讲解 | 关键点是... | 200-300字 |
| 第4段 | 实战应用 | 怎么用呢... | 150-200字 |
| 第5段 | 工具/产品介绍 | 如果你想试试... | 100-150字 |
| 第6段 | CTA引导 | 现在，你可以... | 50-100字 |

> 根据目的可调整为"纯知识普及"（重3-4段）或"营销转化"（重5-6段）

---

## 各段落写作规范

### 第1段：痛点/场景切入

**本段核心任务**：让读者产生"说的就是我"的共鸣，激起阅读兴趣。

**必备要素**：
- 一个具体可感的场景（张三/李四的具体困境，不是抽象描述）
- 读者的内心困惑/焦虑（用第一人称"你"）
- 暗示这个困惑有解决方案（但不要在第一段就揭晓）

**场景库**（按主题选择）：

| 主题 | 场景示例 |
|-----|---------|
| 基金入门 | "你是不是也这样：听说基金能赚钱，但打开账户一看，密密麻麻的基金名字，完全不知道从哪下手..." |
| 指数基金 | "身边都在说定投指数基金，但什么是指数？沪深300又是什么？和主动基金有什么区别？" |
| 账户开户 | "想开个证券账户炒股票，但一想到要跑营业部、要填一堆表格，就一直拖着..." |
| 风险测评 | "买理财之前，那个风险测评问卷到底有什么用？测出来的R3、R4代表什么？" |
| 北向资金 | "每天看财经媒体说'北向资金流入XX亿'，这和我买基金有什么关系？" |

**禁止**：不要用"众所周知"、"想必大家"这类脱离群众的表达

**金句方向**：用场景引发共鸣，如"基金亏钱的原因很简单：你买的时机不对，卖的时机也不对"

**数据要求**：
```
可选：引用真实统计数据（如"Wind数据显示，XX%的新手投资者..."）
```

---

### 第2段：问题本质

**本段核心任务**：用最简单的话解释核心概念，不掉书袋，让小白也能看懂。

**必备要素**：
- 一句话定义（这个工具/指标/概念到底是什么）
- 一个生活化类比（把专业概念类比为日常事物）
- 一个关键澄清（常见的误解是什么）

**类比库**：

| 概念 | 优秀类比 |
|-----|---------|
| 指数 | "把全班同学的平均分，当作衡量班级整体水平的指标" |
| ETF | "像买菜套餐，一个套餐打包了一篮子菜（多只股票），不用自己一样样挑" |
| 定投 | "像每月固定往存钱罐里投硬币，金额固定，长期下来成本会被平摊" |
| PE（市盈率） | "相当于你投入的钱，多少年能回本。PE=20，就是20年回本" |
| 北向资金 | "相当于'聪明钱'，外资机构投资者被认为更理性，他们的动向值得参考" |

**金句方向**：用一句话点明本质，如"ETF的本质是让普通人也能像机构一样一键买入一篮子股票"

**数据要求**：
```
可选：真实历史数据作为佐证
```

---

### 第3段：核心知识讲解

**本段核心任务**：系统性地讲解关键知识点，让读者学完能用。

**必备要素**：
- 2-3个核心知识点（不多不少，关键在于理解而非记忆）
- 每个知识点有：是什么 → 怎么用 → 注意什么
- 至少1个真实案例（张三是怎么用这个工具赚钱/避坑的）
- 1-2个常见错误（新手最容易踩的坑）

**知识讲解公式**：
```
概念 → 定义（1句话） → 怎么用（步骤） → 什么时候用（场景） → 注意事项
```

**常见错误清单**：

| 主题 | 新手常见错误 |
|-----|------------|
| 基金定投 | 认为"定投就是躺着赚钱"，忽视市场周期；亏损时停止定投 |
| 指数基金 | 混淆宽基和窄基；在牛市顶点开始定投 |
| 风险测评 | 随意填写；不理解R3/R4的真正含义 |
| 基金费率 | 只看申购费，忽视管理费/托管费的长期影响 |

**金句方向**：用一句话总结最重要的注意事项，如"最大的坑不是买贵了，而是你根本不知道自己买的是什么"

**数据要求**：
```
可选：费率对比数据、真实收益/亏损案例数据
```

---

### 第4段：实战应用

**本段核心任务**：给读者一个可操作的"第一步"，降低行动门槛。

**必备要素**：
- 一个具体的操作步骤（不是理论，是真的可以去做）
- 具体的数字参考（买入金额、频率、目标点位等）
- 可量化的标准（怎么算成功/失败）
- 时间预期（多久能看到效果）

**操作步骤公式**：
```
第一步：XXX（具体操作）
第二步：XXX（具体操作）
第三步：XXX（具体操作）
目标：XXX个月内，XXX指标达到XXX
```

**参考模板**：

| 场景 | 操作步骤 |
|-----|---------|
| 首次定投 | 第一步：下载券商APP，绑定银行卡；第二步：搜索"沪深300ETF"，设定每月定投1000元；第三步：设置定投日期为发薪日后2天 |
| 首次买债基 | 第一步：风险测评重新做，确认为R3；第二步：在稳健理财区找历史年化4-5%的短债基金；第三步：首次买入1000元持有30天观察 |

**金句方向**：用一句话降低行动门槛，如"不需要等到有很多钱才能开始，100块就可以"

**数据要求**：
```
必须：操作相关的具体数字（金额/频率/收益率参考）
```

---

### 第5段：产品/服务介绍（可选段落）

**本段核心任务**：自然引出券商产品/服务，解决"在哪里做"的问题。

**必备要素**：
- 产品的核心优势（3个以内，不要堆砌卖点）
- 产品适合的人群（什么人最适合）
- 和其他选择的对比（为什么选这个而不是其他）
- 使用限制/风险提示（如有）

**禁止**：
- 不能夸大收益（只能说"历史年化X%"）
- 不能承诺保本
- 不能贬低竞品

**引出方式**（自然不生硬）：
```
如果你看完还是觉得复杂，或者没时间自己研究，可以试试...
XX平台/工具能帮你...
我们（本店）的XX服务，专门为XX人群设计...
```

**金句方向**：用一句话点明核心优势，如"最大的好处是：你不需要懂那么多，我们帮你搞定"

**数据要求**：
```
可选：产品参数（费率/起点/历史收益）
```

---

### 第6段：CTA引导

**本段核心任务**：给读者一个明确的下一步动作，降低行动门槛。

**CTA类型选择**：

| CTA类型 | 适用场景 | 示例 |
|-------|---------|------|
| 扫码开户 | 获客转化 | "扫码低佣开户，附赠一对一投资咨询服务" |
| 关注回复 | 私域沉淀 | "关注后回复'入门'，送你一份《基金小白手册》" |
| 留言互动 | 提升参与 | "你第一次买基金是什么时候？现在怎么样了？评论区聊聊" |
| 预约体验 | 服务转化 | "预约我们的免费投资规划咨询，30分钟帮你理清思路" |

**禁止**：
- 不要用"赶紧点击"、"错过就没有了"等催促话术
- 不要用"仅限前100名"制造虚假紧迫感

**数据要求**：
```
无
```

## 标题规范

**结构**：
```
【新手入门】{核心问题标题}（3分钟搞懂）
【投教】{概念名}是什么？{一句话解答}
【实用】{具体操作}完整指南，看这一篇就够了
```

**字数**：主标题 15-25 字

**情绪**：亲切温暖，像朋友在教你，不是老师在训话

**标题类型选择**：

| 类型 | 示例 |
|-----|------|
| 问题型 | 【新手入门】ETF到底怎么买卖？一文讲清楚 |
| 数字型 | 【实用】5分钟搞懂基金定投，看完就可以开始 |
| 承诺型 | 【投教】看完这篇，基金亏钱的概率降低一半 |
| 场景型 | 【新手入门】月薪5000，如何开始人生第一次基金定投？ |

## 数据使用规范

### 数据使用原则

**能用生活类比就不用数据**——投教内容的读者很多是小白，数据反而增加理解成本。

### 可用数据类型

| 数据类型 | 表达方式 | 用途 |
|---------|---------|------|
| 历史收益率 | "过去10年，沪深300指数年化收益约8%" | 建立信任 |
| 费率对比 | "主动基金平均管理费1.5%，指数基金平均0.5%" | 说服行动 |
| 时间数据 | "定投3年以上，盈利概率超80%" | 增强信心 |
| 比例数据 | "100个投资者，70个亏钱" | 痛点共鸣 |

### 数据来源标注

| 数据类型 | 标注格式 |
|---------|---------|
| 历史收益 | "根据Wind数据，历史收益不代表未来" |
| 费率数据 | "根据基金年报披露" |
| 调研数据 | "根据XX证券投资者调研" |

## 句式规范

### 推荐使用的句式

**亲切引导句**：
```
不用担心，其实很简单...
你可能会问XX，我来一一解答...
不需要你懂XX，只需要知道XX就够了...
很多新手都会犯这个错误，今天我们来避坑...
```

**行动促进句**：
```
其实你现在就可以试着XX，只需要XX步...
只要你有XX元，就可以开始...
现在开户的话，还能享受XX优惠...
```

**对比解释句**：
```
如果你之前买过XX基金，可能会问XX。区别在于：XX
XX和XX的区别，就像XX和XX的区别一样...
```

### 禁止使用的句式

```
❌ 赶紧开户，今天优惠就结束了（虚假紧迫感）
❌ 错过这篇文章，你将损失XX（夸大后果）
❌ 买了这个基金，保证不亏钱（承诺收益）
❌ XX是垃圾，别买！（贬低竞品）
❌ 看完这个你就成为投资高手了（夸大效果）
```

## 词汇规范

### 投教内容高频词

| 术语 | 处理方式 |
|-----|---------|
| 定投 | 必须解释，定期定额投资 |
| ETF | 必须解释，交易所交易基金 |
| 申购/赎回 | 必须解释，买入/卖出 |
| R3/R4 | 必须解释，风险等级 |
| 年化收益 | 必须强调，历史数据不代表未来 |
| 建仓/加仓 | 可解释可不解释 |
| 封闭式/开放式 | 可解释可不解释 |

### 转换词（书面语→口语）

| 书面语 | 口语化表达 |
|-------|-----------|
| 申购 | 买入/购买 |
| 赎回 | 卖出/取出 |
| 收益率 | 赚了多少钱 |
| 风险等级 | 会不会亏钱、亏多少 |
| 资产配置 | 怎么分配钱 |

## 风险提示规范

**位置**：文章结尾单独成段（必须，不可省略）

**格式**：
```
本内容仅供参考，不构成投资建议。投资者据此操作，风险自担。
市场有风险，投资需谨慎。

{如有产品推荐需追加：}
基金有风险，投资需谨慎。历史收益不代表未来表现。
```

**合规检查项**：
- [ ] 无收益承诺（"保证赚钱"类表述）
- [ ] 无保本承诺
- [ ] 风险提示完整
- [ ] 数据有"历史不代表未来"备注

## 质量检查清单

**结构层面**：
- [ ] 第1段有具体场景，读者能"对号入座"
- [ ] 第2段有生活化类比，小白能看懂
- [ ] 第3段有实战操作步骤，不是纯概念
- [ ] 有至少1个"避坑"提醒

**转化层面**：
- [ ] 有明确的下一步动作（CTA）
- [ ] 行动门槛足够低（多少钱/多简单）
- [ ] 没有制造虚假紧迫感

**合规层面**：
- [ ] 无收益承诺
- [ ] 无保本承诺
- [ ] 风险提示完整
- [ ] 数据有"历史不代表未来"备注
- [ ] 无贬低竞品表述

**温度层面**：
- [ ] 像朋友在教你，不是老师在训话
- [ ] 没有"你应该/你必须"的说教感
- [ ] 有真实案例（张三李四）
- [ ] 读者读完后不会感到焦虑，而是感到"我可以试试"
