# 市场评论框架

**适用场景**：每日/每周 A 股市场走势分析，收评、早评、盘后解读。
**核心特征**：数据驱动、事件跟踪、盘面结构解读、资金动向、板块轮动
**篇幅建议**：800-1200 字（短评）/ 1500-2000 字（深度收评）
**内容类型路由权重**：Primary

---

## 段落结构

| 段落 | 功能 | 固定开头语 | 字数范围 |
|:---:|:---|:---|:---:|
| 第1段 | 市场概述 | 昨日A股市场... | 100-150字 |
| 第2段 | 事件影响 | 近期事件层面... | 150-200字 |
| 第3段 | 外围传导 | 关联市场方面... | 100-150字 |
| 第4段 | 技术走势 | 走势结构上... | 100-150字 |
| 第5段 | 资金板块 | 资金与板块... | 150-200字 |
| 第6段 | 后市展望 | 操作思路上... | 100-150字 |

> 短评（800字内）可压缩为4段：市场概述→板块资金→技术解读→操作建议

---

## 各段落写作规范

### 第1段：市场概述

**本段核心任务**：用关键数据勾勒今日市场全貌，给读者一个定量锚点。

**必备要素**：
- 主要指数涨跌幅（上证/深证/创业板/科创板），精确到小数点后2位，带方向（涨/跌）
- 两市成交额（单位：万亿元 or 亿元）
- 涨跌家数对比（上涨/下跌/平盘）
- 今日核心事件一句话

**可选要素**：
- 北向资金流向（如有数据）
- 开盘/收盘点位对比

**金句方向**：用一句话点明今日市场的关键特征，如"指数分化加剧，题材炒作有所降温"

**数据要求**：
```
必须：指数涨跌幅（精确到0.01%）、成交额（精确到亿元）
可选：北向资金、涨跌家数
```

---

### 第2段：事件影响

**本段核心任务**：深度拆解今日最重要的1-2个事件/政策，及其对市场的影响路径。

**必备要素**：
- 事件核心内容（1-2句话说清楚，不复述新闻原文）
- 事件对市场的直接影响（板块/概念/情绪）
- 事件背后的深层逻辑（为什么这件事重要）

**可选要素**：
- 历史对照（类似事件A股曾如何反应）
- 政策背景溯源

**金句方向**：点明事件与表面报道不同的深层含义，如"这个政策调整的真正信号不在于幅度，而在于方向"

**数据要求**：
```
必须：事件相关板块涨跌幅（如有）
可选：历史相似事件的市场反应数据
```

---

### 第3段：外围传导

**本段核心任务**：梳理外围市场（美股/港股/汇率/大宗商品）对A股的传导路径。

**必备要素**：
- 昨夜美股表现（道指/纳指/标普，涨跌幅）
- 港股今日表现（恒生指数，涨跌幅）
- 汇率变动（美元/人民币，离岸价）
- 传导逻辑（外围涨跌如何影响A股风险偏好）

**可选要素**：
- 富时A50表现
- 大宗商品表现（原油/黄金）
- 北向资金盘中流向

**金句方向**：一句话总结外围对A股的边际影响，如"外围波动不改A股中期方向"

**数据要求**：
```
必须：美股涨跌幅（精确到0.01%）、恒生指数涨跌幅
可选：美元/人民币汇率数值、富时A50表现
```

---

### 第4段：技术走势

**本段核心任务**：从技术分析角度解读市场结构，给操作建议提供技术依据。

**必备要素**：
- 关键技术位置（压力位/支撑位）
- 成交量变化（放量/缩量，对比前日）
- 均线系统状态（多头/空头/纠缠）
- 短中期趋势判断

**可选要素**：
- 布林带状态
- MACD柱状图
- K线形态（如锤子线、吞没形态）
- KDJ/RSI 等摆动指标

**金句方向**：用技术语言给出一个方向性判断，如"缩量整固后，若能突破MA5压制，短期或有机会"

**数据要求**：
```
必须：成交量（较前一日缩放比例）、MA5/MA10/MA20位置关系
可选：MACD柱状图方向、KDJ数值、布林带位置
```

---

### 第5段：资金板块

**本段核心任务**：捕捉今日资金动向，找到最强/最弱板块，理解市场结构性特征。

**必备要素**：
- 主力资金净流入/流出板块（TOP3）
- 今日最强概念/主题（1-2个）
- 今日最弱板块（1-2个）
- 板块轮动规律（是否有序轮动还是快速切换）

**可选要素**：
- 涨停家数/跌停家数
- 龙头股表现
- 融资融券余额变化（如有）
- ETF申赎数据（大资金动向）

**金句方向**：用资金语言总结市场结构，如"资金从消费板块向科技迁移的迹象明显"

**数据要求**：
```
必须：资金净流入TOP3板块、涨跌幅超5%的概念板块
可选：涨停家数、龙头股表现
```

---

### 第6段：后市展望

**本段核心任务**：给出有条件、有概率的后市判断，以及操作层面的建议。

**必备要素**：
- 核心判断（看多/看空/看震荡，附概率表述）
- 判断依据（2-3个核心逻辑，不要超过3个）
- 关键观察信号（什么情况证实/证伪你的判断）
- 操作建议（偏向性建议，带概率表述和风险提示）

**可选要素**：
- 明日关注事件（重要数据发布/政策会议）
- 时间窗口判断（如"节前最后3个交易日"）

**金句方向**：用概率语言表达确定性，如"若成交量能维持在1.5万亿以上，大概率仍有上行动能；反之需谨慎"

**数据要求**：
```
必须：无（基于前5段数据的综合判断）
可选：关键支撑/压力位数值
```

## 标题规范

**结构**：
```
【{栏目名}·{文章类型}】{核心观点标题}
——本栏目为"{作者}"观点{收评/早评}
```

**字数**：主标题 15-25 字

**情绪**：冷静克制，用数据说话，不情绪化

**示例**：
```
【年年有鱼·市场评论】节前收官在即，鱼群高低切换待春归
——本栏目为"于晴于理"观点收评

【年年有鱼·市场评论】缩量整固三千点，鱼群分歧待破局
——本栏目为"于晴于理"观点收评
```

## 数据使用规范

### 必需数据

| 数据类型 | 精度要求 | 表达方式 |
|---------|---------|---------|
| 指数涨跌幅 | 精确到0.01% | "+0.09%" 或 "-1.08%" |
| 成交额 | 精确到亿元 | "2.15万亿元" 或 "缩量约300亿元" |
| 板块涨跌幅 | 精确到0.01% | 同上 |
| 资金净流入 | 精确到亿元 | "净流出约50亿元" |

### 表达规范

```
✅ 正确：上证指数微涨0.09%，深证成指下跌0.35%
❌ 错误：上证涨了一点，创业板大跌

✅ 正确：两市成交额2.15万亿元，较前一日缩量约300亿元
❌ 错误：成交额大概两万亿左右

✅ 正确：北向资金净流出约50亿元
❌ 错误：外资跑了
```

### 数据来源标注

| 数据类型 | 标注格式 |
|---------|---------|
| 指数行情 | "根据东方财富数据显示" |
| 资金流向 | "根据东方财富数据显示" |
| 板块资金 | "根据同花顺数据显示" |
| 政策文件 | "根据证监会XX号文件" |

## 句式规范

### 必须使用的句式

**条件判断句**（体现研究的严谨性）：
```
若成交量不能有效放大，突破前高难度较大
若外围市场持续走弱，A股大概率跟随调整
若北向资金持续净流出，需警惕情绪传导
```

**概率性表达**（体现不确定性意识）：
```
大概率维持震荡格局
短期或有机会
仍需观察
值得关注
倾向于
```

### 推荐使用的句式

**对比表达**：
```
较前一日缩量约XX亿元
环比上涨XX%
同比来看...
```

**递进表达**：
```
不仅...而且...
从...到...可以看出
值得注意的是...
```

### 禁止使用的句式

```
❌ 必将大涨、必将大跌
❌ 确定上涨、确定下跌
❌ 一定涨、一定跌
❌ 推荐买入、推荐卖出
❌ 赶紧买、赶紧卖
❌ 满仓梭哈
❌ 抄底、逃顶
❌ 暴涨、暴跌（用"大涨""大跌"代替）
```

## 词汇规范

### 高频专业术语

| 术语 | 含义/用法 |
|-----|---------|
| 缩量/放量 | 成交量较前一日减少/增加 |
| 整固 | 股价在一定区间震荡整理 |
| 突破/站稳 | 价格上穿关键位置/持续高于关键位置 |
| 压制/支撑 | 关键位置阻碍/托住价格 |
| 轮动 | 资金在不同板块间切换 |
| 情绪传导 | 市场情绪从某板块扩散至全市场 |
| 风险偏好 | 投资者愿意承担风险的程度 |
| 增量资金 | 新进入市场的资金 |
| 存量资金 | 已在市场中的资金 |

### 谨慎词使用

```
✅ 推荐：大概率、或、可能、值得关注、建议关注、可以考虑
❌ 禁用：必然、一定、保证、绝对、肯定
```

### 特色比喻词（可选风格）

| 比喻词 | 指代含义 |
|-------|---------|
| 鱼群 | 市场参与者/资金 |
| 鱼尾行情 | 行情末期 |
| 喂鱼 | 引入增量资金 |
| 渔网 | 阻力位/支撑位 |
| 鱼塘 | 特定板块/概念 |

## 风险提示规范

**位置**：文章最后一段（第6段操作建议段）的结尾，或单独成段

**格式**：
```
本内容仅供参考，不构成投资建议。投资者据此操作，风险自担。
市场有风险，投资需谨慎。
```

**合规检查项**：
- [ ] 无绝对化市场预测词汇（必将/确定/一定/必然）
- [ ] 无操作承诺性语句（推荐买入/赶紧买/满仓梭哈）
- [ ] 使用概率性表达（大概率/或/可能）
- [ ] 风险提示完整（不构成投资建议+风险自担）
- [ ] 数据有来源标注

---

## 质量检查清单

初稿完成后逐项核对：

**结构层面**：
- [ ] 第1段包含市场概述（指数+成交额+涨跌家数）
- [ ] 第2段包含事件/政策分析（1-2个核心事件）
- [ ] 第3段包含外围市场传导（美股+港股+汇率）
- [ ] 第4段包含技术走势分析（量价+均线）
- [ ] 第5段包含资金板块（主力板块+轮动）
- [ ] 第6段包含操作建议（有概率表述）

**内容层面**：
- [ ] 全文使用条件判断句（如"若...则..."）≥3处
- [ ] 全文使用概率性表达（大概率/或/可能）≥3处
- [ ] 全文无绝对化预测词汇（必将/确定/一定）
- [ ] 关键数据有来源标注
- [ ] 有至少1句可独立截图的金句

**合规层面**：
- [ ] 风险提示完整（格式正确）
- [ ] 无个股推荐（如提及个股需有充分理由且带风险提示）
- [ ] 无收益承诺（"一定能赚钱"类表述）
- [ ] 无极端表述（"暴跌""崩盘"类）

**篇幅层面**：
- [ ] 短评：800-1200字
- [ ] 深度收评：1500-2000字
- [ ] 各段落长度符合上表建议
