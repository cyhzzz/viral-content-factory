## 标题优化

### 标题规则

- **长度**：20-28 个中文字（微信限制64字符）
- **核心元素**：关键词 + 悬念/情绪/数字
- **策略类型**：

| 策略 | 示例 | 适用场景 |
|-----|------|---------|
| 数字型 | 3个方法、5个技巧、90%的人不知道 | 干货、教程 |
| 信息差 | 你以为...其实...、被忽略的... | 热点解读 |
| 反直觉 | 为什么X反而更好、别再...了 | 观点、对比 |
| 痛点 | 直接戳目标读者的具体问题 | 投教、营销 |
| 悬念 | ...结果出乎意料、...背后的秘密 | 故事、分析 |

### 标题输出

```markdown
### 备选标题

1. **{标题1}**（策略：{数字型}）
2. **{标题2}**（策略：{反直觉}）
3. **{标题3}**（策略：{痛点}）
```

## 摘要优化

### 摘要规则

- **长度**：≤40字
- **要素**：核心关键词 + 价值承诺/悬念
- **注意**：不要重复标题

### 摘要示例

```
原标题：2024年，基金定投必须知道的3件事
摘要：基金亏钱不是因为基金差，是因为方法错。3个关键改变，大概率能提升收益。
```

## 关键词优化

### 关键词规则

- 核心关键词在**前200字**内出现
- 全文自然出现3-5次
- 用同义词/近义词替换，避免堆砌

### 关键词提取

从文章中提取：
1. 核心主题词（1-2个）
2. 延伸关键词（2-3个）
3. 长尾词（1-2个）

---

## 标签推荐

### 标签规则

- **数量**：5个
- **组合**：2行业大词 + 2热点词 + 1长尾词

### 标签格式

```
#行业大词1 #行业大词2 #热点词1 #热点词2 #长尾词
```

---

## 完读率优化

### 段落控制

- 每段 ≤ 150字
- 每3-4段设置钩子（悬念/反转/金句）

### 视觉节奏

- 每400-500字插入配图
- 关键数据/结论用**加粗**
- H2标题有信息量
