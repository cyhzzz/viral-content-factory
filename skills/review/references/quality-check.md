# 内容质量审校规则

> 本文档是审校流程 Step 5.2 的详细操作手册
> 评估初稿的深入浅出程度、吸引力、情绪弧线

---

## 审校维度

| 维度 | 评估内容 | 通过标准 |
|------|---------|---------|
| 深入浅出 | 是否说人话、小白是否读懂 | 专业术语有解释，逻辑清晰 |
| 吸引力 | 开头抓人程度、信息密度 | 开头有钩子，信息量充足 |
| 情绪弧线 | 是否有起伏、有无高潮 | 有节奏感，结尾有力 |

---

## 维度一：深入浅出

### 评估标准

| 检查项 | 标准 | 评估 |
|-------|------|------|
| 专业术语 | 首次出现有解释，或从语境能推断含义 | ✅通过 / ⚠️需改 |
| 逻辑链条 | 结论有依据，因果关系清晰 | ✅通过 / ⚠️需改 |
| 例子支撑 | 抽象观点有具体案例/数据支撑 | ✅通过 / ⚠️需改 |
| 避免掉书袋 | 没有大段落学术表达 | ✅通过 / ⚠️需改 |

### 问题定位格式

```markdown
### 深入浅出

- **评估**：⚠️ 有问题
- **问题1**：
  - 位置：第3段
  - 原文：「{有问题的句子}」
  - 问题：{描述}
- **建议1**：
  - 修改方式：{具体建议}
  - 修改后：「{修改后的句子}」
```

### 常见问题与修改建议

| 问题 | 修改方式 |
|-----|---------|
| 专业术语无解释 | 括号补充解释，或换通俗说法 |
| 因果关系跳跃 | 补充过渡句 |
| 纯理论无案例 | 追加1-2个具体案例 |
| 句式过于复杂 | 拆分为短句 |

---

## 维度二：吸引力

### 评估标准

| 检查项 | 标准 | 评估 |
|-------|------|------|
| 开头钩子 | 前3句有悬念/冲突/好奇心 | ✅通过 / ⚠️需改 |
| 价值承诺 | 读者知道读完后能得到什么 | ✅通过 / ⚠️需改 |
| 信息密度 | 每段有增量信息，无废话段落 | ✅通过 / ⚠️需改 |
| 干货感 | 有可操作的具体建议/知识点 | ✅通过 / ⚠️需改 |

### 问题定位格式

```markdown
### 吸引力

- **评估**：⚠️ 有问题
- **问题1**：开头太平淡
  - 位置：第1段
  - 原文：「{开头原文}」
  - 问题：{描述}
- **建议1**：
  - 修改方式：{具体建议}
  - 修改后：「{修改后的开头}」
```

### 常见问题与修改建议

| 问题 | 修改方式 |
|-----|---------|
| 开头背景铺陈太长 | 删减背景，直切主题 |
| 开头无悬念 | 补一句制造认知落差 |
| 价值承诺缺失 | 开头加一句"读完你会知道..." |
| 中间段落信息重复 | 删除重复，补充新信息 |

---

## 维度三：情绪弧线

### 评估标准

| 检查项 | 标准 | 评估 |
|-------|------|------|
| 开头情绪 | 有情绪起点（好奇/认同/焦虑） | ✅通过 / ⚠️需改 |
| 中间起伏 | 有情绪高低变化，不平铺直叙 | ✅通过 / ⚠️需改 |
| 高潮设计 | 至少一处情绪高峰 | ✅通过 / ⚠️需改 |
| 结尾力量 | 结尾有力，不虎头蛇尾 | ✅通过 / ⚠️需改 |

### 问题定位格式

```markdown
### 情绪弧线

- **评估**：⚠️ 有问题
- **问题1**：结尾无力
  - 位置：最后一段
  - 原文：「{结尾原文}」
  - 问题：{描述}
- **建议1**：
  - 修改方式：{具体建议}
  - 修改后：「{修改后的结尾}」
```

### 常见问题与修改建议

| 问题 | 修改方式 |
|-----|---------|
| 全程平淡无起伏 | 在关键转折处加情绪词 |
| 无情绪高峰 | 在核心观点处加强语气 |
| 虎头蛇尾 | 重写结尾，加强收束感 |
| 情绪突兀 | 增加过渡，让情绪变化自然 |
