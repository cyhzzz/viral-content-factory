# 合规审校规则

> 本文档是审校流程 Step 5.3 的详细操作手册
> 原始来源：`references/compliance-source.md`（申万宏源财富管理事业部AI合规优化助手提示词）

---

## 合规审校输出格式

```markdown
### 合规摘要

{交通信号灯} 【🟢 低风险】/【🟡 中风险】/【🔴 高风险】

核心问题与优化原则（1-2句话）：

### Error（🔴 阻断 — 必须修改）

{每个Error：问题定位 + 修改建议 + 优化示例}

### Warning（🟡 提醒 — 表述过于极端）

{每个Warning：问题定位 + 修改建议}

### Info（🔵 建议 — 供参考）

{每个Info：参考意见}
```

---

## Error（阻断）

必须修改才能发布。

### 1. 提及个股

**问题**：直接提及个股名称

**处理**：必须修改

**修改方式**：
- 个股 → 板块/行业/ETF
- "XX股票" → "XX行业龙头"

**示例**：

| 原句 | 修改后 |
|-----|-------|
| "贵州茅台今日涨3%" | "白酒龙头今日表现亮眼" |
| "建议关注宁德时代" | "建议关注新能源板块" |

---

### 2. 收益承诺

**问题**：承诺收益、保证赚钱

**处理**：必须修改

**禁止词汇**：

| 类别 | 禁用词 |
|-----|-------|
| 市场预测 | 必将大涨、必将大跌、确定上涨、确定下跌、一定涨、一定跌、必然会、必定会 |
| 操作承诺 | 推荐买入、推荐卖出、赶紧买、赶紧卖、满仓梭哈、抄底、逃顶 |

**修改方式**：删除承诺，改为概率性表述

**示例**：

| 原句 | 修改后 |
|-----|-------|
| "买了XX一定涨" | "XX大概率有机会" |
| "赶紧买，不买就错过" | "可以关注" |
| "抄底的时候到了" | "或许可以逢低布局" |

---

### 3. 极端表述

**问题**：过于确定性、极端化词汇

**处理**：必须修改

**禁止词汇**：

| 原词 | 修改后 |
|-----|-------|
| 暴涨 | 大涨 |
| 暴跌 | 大跌 |
| 极好 | 较好 |
| 极差 | 较差 |
| 完全 | 基本 |
| 彻底 | 基本 |
| 只能（操作建议中） | 可以 |
| 必须（操作建议中） | 建议 |

**示例**：

| 原句 | 修改后 |
|-----|-------|
| "市场暴涨" | "市场大涨" |
| "这完全是错误的" | "这可能存在一定问题" |

---

## Warning（提醒）

表述过于极端，建议修改，用户可选择保留。

### 1. 确定性过高

**问题**：缺乏概率缓冲

**推荐词汇**：

| 推荐词 | 用途 |
|-------|------|
| 大概率 | 表示可能性高 |
| 或 | 表示不确定性 |
| 可能 | 表示可能性 |
| 倾向于 | 表示倾向性 |
| 值得关注 | 引起注意 |

**示例**：

| 原句 | 修改后 |
|-----|-------|
| "明天会涨" | "明天大概率会涨" |
| "XX是最好的选择" | "XX或许值得关注" |

---

### 2. 操作建议过强

**问题**：建议过于激进

**推荐词汇**：

| 推荐词 |
|-------|
| 建议关注 |
| 可以考虑 |
| 逢低布局 |
| 控制仓位 |
| 分散风险 |

**示例**：

| 原句 | 修改后 |
|-----|-------|
| "满仓干" | "可以保持适度仓位" |
| "现在必须买" | "现在可以关注" |

---

## Info（建议）

供参考优化，不阻断发布。

| 建议类型 | 说明 |
|---------|------|
| 信息密度 | 某段信息密度可提升 |
| 开头优化 | 开头可更抓人 |
| 金句建议 | 可增加1-2处可传播的金句 |
| 数据标注 | 某数据可补充来源标注 |

---

## 风险提示规范

### 风险提示格式（必须完整）

文章结尾必须包含：

```
本内容仅供参考，不构成投资建议。投资者据此操作，风险自担。
市场有风险，投资需谨慎。
```

### 风险提示检查

- [ ] 风险提示存在
- [ ] 格式完整（不构成投资建议 + 风险自担 + 市场有风险）
- [ ] 位置正确（文章结尾）

---

## 数据来源规范

### 必须标注来源的数据

| 数据类型 | 标注格式 |
|---------|---------|
| 指数行情 | "根据东方财富数据显示" |
| 资金流向 | "根据东方财富数据显示" |
| 政策文件 | "根据证监会XX号文件" |
| 新闻报道 | "根据XX媒体报道" |

### 禁止

- ❌ "据悉"（必须具体来源）
- ❌ "有消息称"（必须核实）
- ❌ 无来源数据（无法核实的数据标注"据市场传闻"）

## 合规检查清单

### 必须通过的检查

- [ ] 无个股提及（Error）
- [ ] 无收益承诺词汇（Error）
- [ ] 无极端表述词汇（Error）
- [ ] 风险提示完整（Error）
- [ ] 数据有来源标注（Error）

### 建议通过的检查

- [ ] 使用概率性表达（Warning）
- [ ] 操作建议用词温和（Warning）
- [ ] 数据来源具体（Warning）

---

## 修订版全文生成规则

1. 读取原始初稿
2. 逐条应用 Error 修改（直接修改原文）
3. 保留 Warning（标注供用户选择）
4. 忽略 Info（仅供参照）
5. 确保风险提示完整
6. 输出融合所有修改的完整版本

### 修订版格式

```markdown
# 修订版全文

{完整文章，含所有Error修改}
（黄色高亮部分为Warning，供用户选择是否保留）
```
