# 用户指南 — 爆款智坊 v2 快速上手

> 本指南面向使用 viral-content-factory 进行内容创作的所有用户。

## 第一次使用（必读）

### 第一步：自动初始化

首次说「写一篇公众号」或任何创作指令时，系统会自动进入两阶段初始化：

**Round 1 — 创建工作空间**
- 询问你想把文件放哪里
- 自动创建 7 个子目录（草稿/已发布/素材/调研/模板/归档）
- 生成 workspace/README.md 说明每个文件夹的用途
- 整个过程约 30 秒

**Round 2 — 录入风格**
- 请你提供 1-3 篇自己写过的满意文章
- 系统分析你的写作风格（语气、句式、用词、结构等）
- 展示分析结果让你确认
- 同时初始化记忆库种子

> 提供的文章越多，风格分析越准确。3篇以上是最佳选择。

### 初始化后

之后每次只需一句话即可开始创作。系统会在后台自动：
- 扫描记忆库获取上下文
- 避免重复选题
- 注入你喜欢的表达方式
- 在适当位置复用金句和素材

## 日常使用

### 写新文章

```
写一篇关于 AI Agent 投资机会的公众号文章
```

系统会自动执行：热点抓取 → 选题 → 素材采集 → 初稿写作 → 审校 → 多平台改写 → 输出

### 基于已有内容改写

```
帮我改写这篇文章：[粘贴文本或URL]
```

系统会拆解原文结构，按你的风格重新改写。

### 交互模式

```
交互模式，写一篇关于效率工具的推文
```

在选题、框架选择等关键节点暂停，等你确认后再继续。

## 让它越来越像你

### 学习修改（最重要）

每次你对初稿做了修改后：

```
学习我的修改
```

系统会分析你的修改模式，更新风格档案。效果：

| 学习次数 | 平均需要手动修改的比例 |
|---------|---------------------|
| 0 次 | ~30% |
| 5 次 | ~15% |
| 20 次 | ~5% |

### 查看学习到了什么

```
看看文章数据怎么样    # 效果复盘
查看我的金句          # 浏览金句库
查看选题库            # 已写/待写的选题
```

### 导入更多范文

随时可以说「导入范文」，提供新的文章让系统进一步学习你的风格。

## 记忆系统说明（v2 新增）

v2 版本新增了结构化记忆系统。以下是各文件的用途：

| 记忆文件 | 存什么 | 你可以 |
|---------|--------|--------|
| 金句库 | 你文章中的精彩句子 | 查看/手动添加 |
| 观点库 | 你的核心观点和立场 | 确认/修正立场 |
| 标题库 | 标题效果和学习记录 | 查看/添加爆款标题参考 |
| 选题库 | 所有候选和已用选题 | 查看/标记废弃 |
| 效果数据 | 文章阅读量和互动数据 | 提供「看看文章数据」触发 |
| 对标账号 | 关注的竞品账号 | 手动添加/查看分析 |
| 内容日历 | 发布排期计划 | 系统自动维护 |
| 反馈记录 | 你所有的修改偏好 | 系统自动积累 |
| 素材库 | 高质量可复用素材 | 查看/手动补充 |
| 读者画像 | 目标读者特征 | 确认/修正 |

这些文件都在 `memory/` 目录下，你可以随时查看和编辑。

## 输出文件位置

### v2 默认输出到 workspace/

```
workspace/
├── drafts/YYYY-MM-DD_标题/
│   ├── draft.md              # 审校后初稿
│   ├── wechat/article.html    # 微信 HTML
│   ├── wechat/article.txt     # 微信纯文本
│   ├── xhs/note.md           # 小红书笔记
│   ├── weibo/post.md         # 微博文案
│   └── ...                   # 其他平台
└── published/                # 确认发布后移到这里
```

### 兼容旧版 output/ 目录

如果 config.yaml 中没有配置 workspace 路径，仍然输出到 `output/platforms/`。

## 常见问题

**Q: 可以只在一个平台上发布吗？**

A: 可以。在 Step 0.5 平台确认时告诉系统只需要某个平台即可。

**Q: 风格录入后可以修改吗？**

A: 可以。随时说「重新设置风格」或「修改风格」来调整。「学习我的修改」会让风格持续进化。

**Q: 记忆库文件太多，哪些最重要？**

A: `feedback.md`（反馈记录）优先级最高——它直接影响每次的创作风格。其次是 `titles.md`（标题学习）和 `golden-sentences.md`（金句库）。

**Q: 如何让标题更好？**

A: 系统会自动从你的历史数据中学习标题偏好。你也可以在 `memory/titles.md` 的「对标爆款标题库」中添加你喜欢的爆款标题作为参考。

**Q: 支持哪些平台？**

A: 微信公众号、小红书、微博、知乎、今日头条、抖音（分镜脚本）、哔哩哔哩（分P脚本）。

**Q: 生成的文章可以直接发吗？**

A: 建议至少花 3-5 分钟在编辑锚点处加入你自己的话。文章中会有 2-3 处 `<!-- ✏️ 编辑建议 -->` 标记。
