## 模块职责表

| 模块 | 文件位置 | 职责 | 输入 | 输出 |
|------|---------|------|------|------|
| **主管道** | SKILL.md | 流程编排、入口分流、步骤协调 | 用户指令 | 各步骤调度 |
| **项目初始化** | SKILL.md Init R1/R2 | 创建工作空间、录入风格 | 用户确认 | workspace/ + style.yaml |
| **风格学习** | skills/style-learning | 从范文提取8维风格 | 范文文本 | style.yaml |
| **框架选择** | skills/content-type-framework | 根据内容类型选骨架 | 话题+类型 | 框架文件路径 |
| **拆解改写** | skills/rewrite | 解构外部素材 | URL/文本 | 结构化分析报告 |
| **审校** | skills/review | 三维质量检查 | 初稿 | 问题清单+建议 |
| **平台适配** | skills/platform-adaptation | 多平台格式改写 | 初稿+平台列表 | 7平台文件 |
| **记忆系统** | memory/*.md | 资产沉淀与复用 | 每次创作产出 | 结构化记忆条目 |
| **风格飞轮** | learn-edits.md + playbook.md | 编辑学习→规则化 | draft vs final | playbook 规则 |
| **排版引擎** | toolkit/ | Markdown→微信HTML | Markdown | 内联样式HTML |
| **视觉AI** | toolkit/image_gen.py | AI配图生成 | 描述词 | 图片文件 |
| **热点抓取** | scripts/fetch_hotspots.py | 多平台热搜聚合 | --limit N | 热点JSON |
| **SEO分析** | scripts/seo_keywords.py | 关键词量化评分 | 关键词列表 | SEO评分 |

## 质量闸门表

| 闸门 | 位置 | 检查项 | 通过条件 | 不通过处理 |
|------|------|--------|---------|-----------|
| G1 | Init R2 后 | 风格档案存在 | style.yaml 有效 | 强制进入 Onboard |
| G2 | Step 1.1 | 环境就绪 | Python依赖+配置齐全 | 逐项引导创建 |
| G3 | Step 3.3 | 初稿自检 | 禁用词=0, 句长方差合格 | 修复后继续 |
| G4 | Step 4 | 审校通过 | Error=0 | 必须修复才可下一步 |
| G5 | Step 7 | 记忆写入 | 关键文件已更新 | 不阻断主流程，下次补写 |

## 配置层级

```
config.yaml              ← API密钥等运行时配置（用户填写）
style.yaml               ← 风格档案（Init R2 生成/用户可调）
writing-config.yaml      ← 写作参数（字数/密度/风格强度等）
playbook.md              ← 飞轮学习的硬性规则（confidence ≥5 生效）
memory/feedback.md       ← 用户原始反馈偏好（最高优先级覆盖）
memory/titles.md         ← 标题偏好画像（影响标题生成）
memory/audience.md       ← 读者画像（影响语气和用词）

优先级（高→低）：
feedback.md > playbook(conf≥5) > style.yaml > personas/默认值
```

## 错误处理策略

| 错误级别 | 场景 | 处理方式 | 是否阻断 |
|---------|------|---------|---------|
| FATAL | workspace 无法创建 | 回退到 output/ 目录 | 否（降级） |
| FATAL | style.yaml 无法生成 | 使用默认人格包 + 明确警告 | 否（降级+警告） |
| ERROR | 热点脚本失败 | WebSearch 替代 | 否 |
| ERROR | 审校 Error 未解决 | 断点等待用户处理 | 是 |
| WARNING | 单平台改写失败 | 记录失败，其他平台继续 | 否 |
| INFO | 记忆写入失败 | 下次启动时补写 | 否 |
| INFO | 配置项缺失 | 使用默认值 + 日志提示 | 否 |
